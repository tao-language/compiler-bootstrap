# Grammar-Derived Formatter Plan

> **Goal**: Generate formatter infrastructure from grammar (precedence, layout hints) while keeping manual format function
> **Status**: 📋 Planned
> **Date**: March 2025

---

## Overview

This plan implements a **hybrid approach**: generate formatter infrastructure (precedence table, layout hints) from grammar, but keep manual `format_expr` function for control and exhaustiveness checking.

**Key Principle**: Precedence defined ONCE in grammar, reused by formatter via generated lookup table.

**Estimated Effort**: 3-4 days
**Expected Benefit**: 70-80% reduction in formatter boilerplate, zero precedence duplication

---

## Architecture

### Current Problem

**Precedence duplicated**:
```gleam
// Grammar defines precedence
grammar.left_assoc("Expr", "Term", [
  grammar.op("+", Add, 10, ...),  // ← Precedence 10
  grammar.op("*", Mul, 20, ...),  // ← Precedence 20
], 10)

// Formatter duplicates precedence (CAN DRIFT!)
fn format_expr(ast: Expr, parent_prec: Int) -> Doc {
  case ast {
    Add(l, r) -> format_binop(..., 10, parent_prec)  // ← Must match grammar
    Mul(l, r) -> format_binop(..., 20, parent_prec)  // ← Must match grammar
  }
}
```

**Risk**: As grammar evolves, formatter precedence can drift.

---

### Proposed Solution

**Generate precedence table from grammar**:
```gleam
// Generated by: gleam run generate-formatter-metadata --grammar calc_grammar
// DO NOT EDIT - Generated from grammar

pub fn get_operator_precedence(op: String) -> Option(Int) {
  case op {
    "add" -> Ok(10)
    "sub" -> Ok(10)
    "mul" -> Ok(20)
    "div" -> Ok(20)
    _ -> Error(Nil)
  }
}

pub fn get_operator_layout(op: String) -> Option(OperatorLayout) {
  case op {
    "add" -> Ok(default_op_layout("+"))
    "sub" -> Ok(default_op_layout("-"))
    "mul" -> Ok(default_op_layout("*"))
    "div" -> Ok(default_op_layout("/"))
    _ -> Error(Nil)
  }
}

pub fn get_constructor_precedence(ast: Expr) -> Int {
  case ast {
    Add(_, _) -> 10
    Sub(_, _) -> 10
    Mul(_, _) -> 20
    Div(_, _) -> 20
    Int(_) -> 100  // Literals have highest precedence
  }
}
```

**Formatter uses generated table**:
```gleam
// Manual but simple - no precedence duplication!
fn format_expr(ast: Expr, parent_prec: Int) -> Doc {
  case ast {
    Add(l, r) ->
      formatter.format_binop_auto(
        format_expr,  // Pass formatter recursively
        l,
        r,
        "add",  // ← Operator name, not precedence!
        parent_prec,
      )
    Int(n) -> formatter.text(int.to_string(n))
  }
}
```

**Key insight**: Formatter passes operator **name**, not precedence. Precedence lookup happens in combinator.

#### 1.1 Add Metadata Extraction API

**File**: `src/syntax/grammar.gleam`

**Implementation**:

```gleam
// ============================================================================
// FORMATTER METADATA EXTRACTION
// ============================================================================

/// Extract operator precedence table from grammar
///
/// Returns a function that can lookup precedence by operator name
pub fn extract_precedence_table(
  grammar: Grammar(a),
) -> fn(String) -> Result(Int, Nil) {
  let operators = grammar.operators
  
  fn(op_name: String) -> Result(Int, Nil) {
    case dict.get(operators, op_name) {
      Ok(op) -> Ok(op.precedence)
      Error(_) -> Error(Nil)
    }
  }
}

/// Extract operator layout table from grammar
///
/// Returns a function that can lookup layout by operator name
pub fn extract_layout_table(
  grammar: Grammar(a),
) -> fn(String) -> Result(OperatorLayout, Nil) {
  let operators = grammar.operators
  
  fn(op_name: String) -> Result(OperatorLayout, Nil) {
    case dict.get(operators, op_name) {
      Ok(op) -> Ok(op.layout)
      Error(_) -> Error(Nil)
    }
  }
}

/// Extract constructor precedence from grammar
///
/// Returns a function that can lookup precedence by constructor name
pub fn extract_constructor_precedence(
  grammar: Grammar(a),
  constructor_map: Dict(String, String),  // constructor → operator name
) -> fn(String) -> Result(Int, Nil) {
  let precedence_table = extract_precedence_table(grammar)
  
  fn(ctor_name: String) -> Result(Int, Nil) {
    case dict.get(constructor_map, ctor_name) {
      Ok(op_name) -> precedence_table(op_name)
      Error(_) -> Error(Nil)
    }
  }
}
```

// ============================================================================
// FORMATTER METADATA CODE GENERATOR
// ============================================================================
import gleam/dict.{type Dict}
import gleam/io
import gleam/result
import gleam/string
import simplifile
import syntax/grammar

pub type GenerateError {
  GrammarError(message: String)
  FileError(simplifile.FileError)
}

/// Generate formatter metadata module from grammar
///
/// Usage: gleam run generate-formatter-metadata --grammar calc_grammar --output src/mylang/formatter_metadata.gleam
pub fn generate_formatter_metadata(
  grammar_module: String,
  output_path: String,
) -> Result(Nil, GenerateError) {
  // Load grammar (via reflection or hardcoded mapping)
  let grammar = get_grammar(grammar_module)
    |> result.map_error(GenerateError.GrammarError)
  
  // Generate constructor → operator name mapping
  let constructor_map = build_constructor_map(grammar)
  
  // Generate metadata module
  let metadata_code = generate_metadata_code(grammar, constructor_map)
  
  // Write to output file
  simplifile.write(output_path, metadata_code)
    |> result.map_error(GenerateError.FileError)
  
  io.println("Generated formatter metadata: " <> output_path)
  Ok(Nil)
}

fn get_grammar(grammar_module: String) -> Result(grammar.Grammar(a), String) {
  // For now, hardcoded mapping
  // Future: use Gleam reflection
  case grammar_module {
    "calc_grammar" -> Ok(calc_grammar())
    _ -> Error("Unknown grammar: " <> grammar_module)
  }
}

fn build_constructor_map(grammar: grammar.Grammar(a)) -> Dict(String, String) {
  // Map constructor names to operator names
  // e.g., "Add" → "add", "Mul" → "mul"
  grammar.operators
  |> dict.fold(dict.new(), fn(acc, op_name, op) {
    let ctor_name = string.capitalize(op.constructor_name)
    dict.insert(acc, ctor_name, op_name)
  })
}

fn generate_metadata_code(
  grammar: grammar.Grammar(a),
  constructor_map: Dict(String, String),
) -> String {
  let precedence_table = generate_precedence_table(grammar)
  let layout_table = generate_layout_table(grammar)
  let ctor_prec = generate_constructor_precedence(grammar, constructor_map)
  
  "
// Generated by: gleam run generate-formatter-metadata
// DO NOT EDIT - Generated from grammar

import gleam/option.{type Option, None, Some}
import syntax/grammar.{type OperatorLayout}

/// Get operator precedence by name
pub fn get_operator_precedence(op: String) -> Option(Int) {
" <> precedence_table <> "
}

/// Get operator layout by name
pub fn get_operator_layout(op: String) -> Option(OperatorLayout) {
" <> layout_table <> "
}

/// Get precedence for AST constructor
pub fn get_constructor_precedence(ctor: String) -> Option(Int) {
" <> ctor_prec <> "
}
"
}

fn generate_precedence_table(grammar: grammar.Grammar(a)) -> String {
  grammar.operators
  |> dict.fold([], fn(acc, op_name, op) {
    ["    \"" <> op_name <> "\" -> Some(" <> int.to_string(op.precedence) <> ")", ..acc]
  })
  |> string.join("\n")
  |> fn(cases) { "  case op {\n" <> cases <> "\n    _ -> None\n  }" }
}

fn generate_layout_table(grammar: grammar.Grammar(a)) -> String {
  grammar.operators
  |> dict.fold([], fn(acc, op_name, op) {
    let layout_str = operator_layout_to_string(op.layout)
    ["    \"" <> op_name <> "\" -> Some(" <> layout_str <> ")", ..acc]
  })
  |> string.join("\n")
  |> fn(cases) { "  case op {\n" <> cases <> "\n    _ -> None\n  }" }
}

fn operator_layout_to_string(layout: grammar.OperatorLayout) -> String {
  let grammar.OperatorLayout(separator, break_before, break_after, indent_rhs) = layout
  "grammar.OperatorLayout(\"" <> separator <> "\", "
  <> bool.to_string(break_before) <> ", "
  <> bool.to_string(break_after) <> ", "
  <> bool.to_string(indent_rhs) <> ")"
}

fn generate_constructor_precedence(
  grammar: grammar.Grammar(a),
  constructor_map: Dict(String, String),
) -> String {
  let precedence_table = grammar.extract_precedence_table(grammar)
  
  constructor_map
  |> dict.fold([], fn(acc, ctor_name, op_name) {
    case precedence_table(op_name) {
      Ok(prec) -> ["    \"" <> ctor_name <> "\" -> Some(" <> int.to_string(prec) <> ")", ..acc]
      Error(_) -> acc
    }
  })
  |> string.join("\n")
  |> fn(cases) { "  case ctor {\n" <> cases <> "\n    _ -> None\n  }" }
}
```

#### 2.1 Add Metadata-Aware Combinators

**File**: `src/syntax/formatter.gleam`

**Implementation**:

```gleam
// ============================================================================
// METADATA-AWARE FORMATTER COMBINATORS
// ============================================================================

/// Format binary operator with automatic precedence lookup
///
/// Instead of passing precedence manually, pass operator name.
/// Precedence is looked up from generated metadata table.
///
/// Example:
/// ```gleam
/// fn format_expr(ast: Expr, parent_prec: Int) -> Doc {
///   case ast {
///     Add(l, r) ->
///       formatter.format_binop_auto(
///         format_expr,  // Recursive formatter
///         l,
///         r,
///         "add",  // ← Operator name (not precedence!)
///         parent_prec,
///       )
///   }
/// }
/// ```
pub fn format_binop_auto(
  format_fn: fn(a, Int) -> Doc,  // Recursive formatter
  left: a,
  right: a,
  op_name: String,
  parent_prec: Int,
) -> Doc {
  let prec = get_operator_precedence(op_name) |> option.unwrap(0)
  let layout = get_operator_layout(op_name) |> option.unwrap(default_op_layout(" "))
  
  let left_doc = format_fn(left, prec)
  let right_doc = format_fn(right, prec + 1)
  
  let separator = case layout.break_before {
    True -> concat([hardline, text(layout.separator)])
    False -> text(layout.separator)
  }
  
  let right_indent = case layout.indent_rhs {
    True -> 2
    False -> 0
  }
  
  let doc = concat([
    left_doc,
    separator,
    case layout.break_after {
      True -> line()
      False -> empty()
    },
    nest(right_indent, right_doc),
  ])
  
  case prec < parent_prec {
    True -> parens(doc)
    False -> doc
  }
}

/// Format unary operator with automatic precedence lookup
pub fn format_unary_auto(
  format_fn: fn(a, Int) -> Doc,
  operand: a,
  op_name: String,
  parent_prec: Int,
  is_prefix: Bool,
) -> Doc {
  let prec = get_operator_precedence(op_name) |> option.unwrap(90)
  let operand_doc = format_fn(operand, prec)
  
  let doc = case is_prefix {
    True -> concat([text(op_name), operand_doc])
    False -> concat([operand_doc, text(op_name)])
  }
  
  case prec < parent_prec {
    True -> parens(doc)
    False -> doc
  }
}

/// Format with multiple layout strategies (try flat, then break)
///
/// Example:
/// ```gleam
/// formatter.format_with_strategies([
///   fn() { format_inline(expr) },      // Try inline first
///   fn() { format_partial_break(expr) }, // Then partial break
///   fn() { format_full_break(expr) },    // Finally full break
/// ])
/// ```
pub fn format_with_strategies(
  strategies: List(fn() -> Doc),
  width: Int,
) -> Doc {
  case strategies {
    [] -> empty
    [first, ..rest] -> {
      let first_doc = first()
      case fits(first_doc, width) {
        True -> first_doc
        False -> format_with_strategies(rest, width)
      }
    }
  }
}

/// Format inline (no breaks)
pub fn format_inline(format_fn: fn(a) -> Doc, value: a) -> Doc {
  group(format_fn(value))
}

/// Format with soft breaks (break if needed)
pub fn format_soft_break(format_fn: fn(a, LayoutHint) -> Doc, value: a) -> Doc {
  format_fn(value, SoftBreak)
}

/// Format with hard breaks (always break)
pub fn format_hard_break(format_fn: fn(a, LayoutHint) -> Doc, value: a) -> Doc {
  format_fn(value, HardBreak)
}

/// Format record with automatic layout
///
/// Tries inline first, then breaks if too long
pub fn format_record_auto(
  fields: List(#(String, Doc)),
  width: Int,
) -> Doc {
  format_with_strategies([
    fn() { format_record_inline(fields) },
    fn() { format_record_break(fields) },
  ], width)
}

fn format_record_inline(fields: List(#(String, Doc))) -> Doc {
  let field_docs = fields |> list.map(fn(f) {
    let #(name, value) = f
    concat([text(name), text(": "), value])
  })
  format_wrapped("{ ", format_list(field_docs, text(", ")), " }")
}

fn format_record_break(fields: List(#(String, Doc))) -> Doc {
  let field_docs = fields |> list.map(fn(f) {
    let #(name, value) = f
    concat([text("  "), text(name), text(": "), value])
  })
  format_wrapped("{\n", format_list(field_docs, text(",\n")), "\n}")
}
```

import syntax/formatter

pub fn format(ast: Expr) -> String {
  format_expr(ast, 0) |> formatter.render(80)

fn format_expr(ast: Expr, parent_prec: Int) -> Doc {
  case ast {
    Int(n) -> formatter.text(int.to_string(n))

## Why This Works

### Solves All Original Problems

| Problem | Solution |
|---------|----------|
| Precedence duplication | Generated metadata table from grammar |
| Layout duplication | Generated layout table from grammar |
| Deconstructor complexity | Single manual pattern match (exhaustive) |
| AST structure mismatch | Constructor → operator mapping |
| No pretty-printing | Uses `src/syntax/formatter.gleam` combinators |

### Maintains Benefits

| Benefit | How |
|---------|-----|
| Control | Manual `format_expr` function |
| Exhaustiveness | Single pattern match, compiler checks |
| Customization | Can override any formatting |
| Pretty-printing | Uses document algebra under the hood |
